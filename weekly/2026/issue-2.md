# 技术周刊 第2期

## 📚 精选文章

1. **[React Compiler：让组件性能优化更自动化]**
   - 关注 React Compiler 的编译期优化思路，理解它如何减少手写 `useMemo`、`useCallback` 的负担
   - 适合正在学习 React 性能优化、组件渲染机制的同学阅读
   - [阅读全文](https://react.dev/learn/react-compiler)

2. **[TypeScript Handbook：重新梳理类型系统基础]**
   - 从类型收窄、泛型、工具类型到模块组织，系统复习 TypeScript 的核心能力
   - 对写全栈项目、维护大型前端代码都很有帮助
   - [阅读全文](https://www.typescriptlang.org/docs/handbook/intro.html)

3. **[Vite Guide：现代前端开发体验的核心工具]**
   - 了解 Vite 的开发服务器、构建优化、插件机制和框架集成方式
   - 适合作为 React / Vue / Svelte 项目的构建工具入门资料
   - [阅读全文](https://vite.dev/guide/)

## 🛠️ 开源项目

1. **[Astro]**
   - 技术栈：JavaScript, TypeScript, Islands Architecture
   - 简介：面向内容站点、博客和作品集的现代 Web 框架，支持部分水合和多框架组件集成
   - [GitHub](https://github.com/withastro/astro)

2. **[Biome]**
   - 技术栈：Rust, JavaScript, TypeScript
   - 简介：高速的格式化、Lint 和代码质量工具，可作为 Prettier / ESLint 工作流的补充或替代方案
   - [GitHub](https://github.com/biomejs/biome)

3. **[Rspack]**
   - 技术栈：Rust, JavaScript
   - 简介：兼容 webpack 生态的高性能构建工具，适合关注大型前端项目构建速度的开发者
   - [GitHub](https://github.com/web-infra-dev/rspack)
